# Nexa AI Assistant

Nexa AI Assistant is a Python-based desktop voice assistant designed to simplify everyday computer tasks through voice commands. The project is being developed as a learning initiative focused on artificial intelligence, automation, and software development.

## Features

### Voice Interaction

* Speech recognition using microphone input
* Text-to-speech responses
* Hands-free command execution

### Multi Wake Word Support

* Jarvis
* Alexa
* Nexa

### Productivity Commands

* Open Google
* Open YouTube
* Open ChatGPT
* Open WhatsApp Web
* Open Chrome
* Open VS Code
* Open File Explorer
* Open Calculator
* Open Notepad

### Information Services

* Current Time
* Current Date
* User Identification

### Memory System

* Store custom memories
* Recall saved information
* Clear stored memories

## Technologies Used

* Python 3
* SpeechRecognition
* Pyttsx3
* Keyboard
* WebBrowser
* OS Module

## Project Goals

The long-term vision of Nexa AI Assistant is to evolve into a fully featured personal AI assistant capable of:

* Natural language conversations
* AI-powered responses
* YouTube media control
* Spotify integration
* Smart desktop automation
* Task scheduling
* Note management
* Graphical User Interface (GUI)
* Advanced memory management

## Installation

1. Clone the repository
2. Install required Python packages
3. Run `nexa.py`
4. Press `F8` to activate the assistant
5. Start interacting using a supported wake word
